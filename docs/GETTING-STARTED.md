# Getting Started

## Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- Internet connection (for Google Fonts CDN and external links)

---

## View the GitHub Profile

The primary profile is the `README.md` — it auto-displays on your GitHub profile page:

```
https://github.com/harshvardhankulkarni
```

No setup required. The README renders automatically on the profile.

---

## View the Portfolio Site

The animated portfolio site is hosted on GitHub Pages:

```
https://harshvardhankulkarni.github.io/harshvardhankulkarni/
```

Open the URL in any browser. The site loads instantly — no build, no install, no server.

---

## Run Locally

1. Clone or download the repository:
   ```powershell
   git clone https://github.com/harshvardhankulkarni/harshvardhankulkarni.git
   ```
2. Open `index.html` in your browser:
   - Double-click the file in File Explorer, OR
   - Drag it into an open browser window

That's it. The site is fully self-contained. No build tools, no package manager, no server required.

---

## Verify It Works

1. The page loads with a dark background and animated gradient orbs.
2. The typing effect cycles through role descriptors in the hero.
3. Scrolling reveals each section with fade-in animations.
4. Navigation links scroll smoothly to each section.
5. All external links open correctly (GitHub repos, LinkedIn, portfolio).

---

## No Dependencies

This project has zero runtime dependencies:
- Zero npm packages
- Zero Python packages
- Zero Ruby gems
- Zero build tools

The only external resource is the Google Fonts CDN for the Inter typeface.
